To run this code:</br>
1. Download the P5 Serial Control app. Link: https://github.com/vanevery/p5.serialcontrol/releases</br>
2. Add Hx711 to your included libraries in Arduino. Go to Sketch > Include Library > Add .ZIP Library </br> 
3. Set up an Arduino/breadboard circuit with a load cell, amplifier, potentiometer and two SPDT switches according to the schematic here: https://www.andreakang.me/blog/2017/10/9/week-4-physical-computing
You can skip the LEDs.</br>
4. Have the Serial Control open while running.</br>
